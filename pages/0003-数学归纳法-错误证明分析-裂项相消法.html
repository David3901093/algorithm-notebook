📚 数学归纳法错误证明分析 —— 裂项相消法

题目

题目 3. [18]

下面定理的归纳法证明表面上是正确的，但是由于某种原因，当 $n=6$ 时，等式左端给出 $\frac{1}{2} + \frac{1}{6} + \frac{1}{12} + \frac{1}{20} + \frac{1}{30} = \frac{5}{6}$，而右端给出 $\frac{3}{2} - \frac{1}{6} = \frac{4}{3}$。你能找出错误吗？

定理：

$$\frac{1}{1 \times 2} + \frac{1}{2 \times 3} + \cdots + \frac{1}{(n-1) \times n} = \frac{3}{2} - \frac{1}{n}$$

证明： 我们对 $n$ 用归纳法。对于 $n=1$，显然 $\frac{3}{2} - \frac{1}{n} = \frac{1}{(1 \times 2)}$。假设定理对于 $n$ 为真，那么

$$\begin{aligned} &\frac{1}{1 \times 2} + \cdots + \frac{1}{(n-1) \times n} + \frac{1}{n \times (n+1)} \\ &= \frac{3}{2} - \frac{1}{n} + \frac{1}{n(n+1)} \\ &= \frac{3}{2} - \frac{1}{n} + \left(\frac{1}{n} - \frac{1}{n+1}\right) \\ &= \frac{3}{2} - \frac{1}{n+1} \end{aligned}$$

🎯 解题思路

这道题是一道经典的数学归纳法陷阱题！表面上看起来每一步推导都完美无缺，但结果却和实际计算不符。

关键问题在于：数学归纳法的两个步骤缺一不可——基础情形验证错了，后面再漂亮的推导都是空中楼阁！

我们不仅要找出错误，还要用正确的方法（裂项相消法）求出真正的公式。

🔍 错误分析

❌ 核心错误：基础情形（$n=1$）验证不成立！

1 验证 $n=1$ 时的真实情况

当 $n=1$ 时：

  • 左边：$\sum_{i=1}^{0} \frac{1}{i(i+1)}$ → 这是一个空和（没有项），应该等于 $0$
  • 右边：$\frac{3}{2} - \frac{1}{1} = \frac{1}{2}$

❌ 左边 ($0$) ≠ 右边 ($\frac{1}{2}$)，基础情形根本不成立！

2 证明中的"障眼法"

证明中写道："对于 $n=1$，显然 $\frac{3}{2} - \frac{1}{n} = \frac{1}{1 \times 2}$"

这句话偷换了概念——它把右边的值强行等于了数列中某一项的值，而不是等于整个左边求和的结果。

就像说"对于 $n=1$，显然 $100 = 1$"一样荒谬！

3 为什么归纳步骤看起来是对的？

证明中的归纳步骤本身在代数运算上没有错误：

  • 如果假设对 $n$ 成立（即左边 = $\frac{3}{2} - \frac{1}{n}$）
  • 那么对 $n+1$ 也成立（即左边 = $\frac{3}{2} - \frac{1}{n+1}$）

但是——多米诺骨牌的第一块就没有被推倒！ 后面的推理再漂亮也没有意义。

💡 多米诺骨牌类比

数学归纳法就像推多米诺骨牌：

  1. 基础情形 = 推倒第一块骨牌 ✅/❌
  2. 归纳步骤 = 确保前一块倒下时，后一块也会倒下 ✅

如果第一步就失败了，第二步设计得再精巧也毫无用处！

✅ 正确解法：裂项相消法

1 裂项技巧

观察每一项的形式 $\frac{1}{k(k+1)}$，我们可以把它"拆开"：

$$\frac{1}{k(k+1)} = \frac{1}{k} - \frac{1}{k+1}$$

这个等式可以通过通分来验证： $$\frac{1}{k} - \frac{1}{k+1} = \frac{(k+1) - k}{k(k+1)} = \frac{1}{k(k+1)}$$ ✅

2 代入原式，观察"消消乐"

$$\begin{aligned} &\frac{1}{1 \times 2} + \frac{1}{2 \times 3} + \cdots + \frac{1}{(n-1) \times n} \\[6pt] &= \left(1 - \frac{1}{2}\right) + \left(\frac{1}{2} - \frac{1}{3}\right) + \left(\frac{1}{3} - \frac{1}{4}\right) + \cdots + \left(\frac{1}{n-1} - \frac{1}{n}\right) \end{aligned}$$

3 中间项全部抵消！

仔细看上面的展开式，你会发现一个神奇的现象：

$$\cancel{-\frac{1}{2}} + \cancel{\frac{1}{2}} + \cancel{-\frac{1}{3}} + \cancel{\frac{1}{3}} + \cdots + \cancel{-\frac{1}{n-1}} + \cancel{\frac{1}{n-1}}$$

所有的中间项都成对抵消了！只剩下第一项的首部和最后一项的尾部：

$$= 1 - \frac{1}{n} = \frac{n-1}{n}$$

📝 用数学归纳法验证正确公式

正确的公式是：$\displaystyle\frac{1}{1 \times 2} + \frac{1}{2 \times 3} + \cdots + \frac{1}{(n-1) \times n} = \frac{n-1}{n}$

1 基础情形（$n=1$）

左边 = 空和 = $0$，右边 = $\frac{1-1}{1} = 0$ ✅

2 归纳假设

假设 $n=k$ 时成立：$\displaystyle\sum_{i=1}^{k-1}\frac{1}{i(i+1)} = \frac{k-1}{k}$

3 归纳步骤（$k \to k+1$）

$$\begin{aligned} \text{左边}_{k+1} &= \frac{k-1}{k} + \frac{1}{k(k+1)} = \frac{(k-1)(k+1)+1}{k(k+1)} \\ &= \frac{k^2-1+1}{k(k+1)} = \frac{k^2}{k(k+1)} = \frac{k}{k+1} \\ &= \frac{(k+1)-1}{k+1} = \text{右边}_{k+1} \quad ✅ \end{aligned}$$

✅ 最终答案

答案

错误原因：基础情形（$n=1$）验证错误。当 $n=1$ 时，左边为空和等于 $0$，但右边为 $\frac{1}{2}$，两者不相等。

正确公式：

$$\boxed{\frac{1}{1 \times 2} + \frac{1}{2 \times 3} + \cdots + \frac{1}{(n-1) \times n} = 1 - \frac{1}{n} = \frac{n-1}{n}}$$

验证：当 $n=6$ 时，左边 $= 1 - \frac{1}{6} = \frac{5}{6}$ ✅ 与题目给出的实际计算一致！

💡 知识拓展

🏛️ 历史故事：裂项相消法的起源

"裂项相消"（Telescoping Series）的思想可以追溯到古希腊时期。阿基米德在计算抛物线弓形面积时就使用了类似的方法，把无穷级数中的项两两配对抵消。

到了 17 世纪，莱布尼茨和牛顿在发展微积分时大量使用了这种技巧。莱布尼茨曾说过："数学的精髓在于发现隐藏在对称中的简洁。"

裂项相消法正是这种思想的完美体现——看似复杂的求和，通过巧妙的拆分，中间全部消失，只留下最简洁的结果！

🔗 更一般的裂项公式

对于形如 $\frac{1}{k(k+d)}$ 的项（$d$ 为正常数）：

$$\frac{1}{k(k+d)} = \frac{1}{d}\left(\frac{1}{k} - \frac{1}{k+d}\right)$$

本题是 $d=1$ 的特例。记住这个通用公式，以后遇到类似的题目就能秒杀了！
