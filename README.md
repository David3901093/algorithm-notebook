# 算法学习之旅 | Algorithm Journey

> 🚀 从零开始，系统掌握算法思维

## 📖 项目简介

这是一个互动式算法学习笔记网站，采用 HTML + MathJax + JavaScript 构建，包含完整的数学推导、代码示例和互动式练习题。每个笔记都是一个独立的 HTML 文件，内嵌 LaTeX 公式渲染、练习题系统和即时反馈功能。

## 🌐 访问方式

1. **本地访问**：直接用浏览器打开 `index.html` 文件
2. **笔记浏览**：点击导航页面中的任意笔记即可跳转到详细内容
3. **搜索功能**：使用顶部搜索框快速查找笔记

## ✨ 特色功能

### 1. LaTeX 公式渲染
- 使用 MathJax 3.x 渲染数学公式
- 支持行内公式 `$...$` 和块级公式 `$$...$$`
- 自动扫描整个文档并渲染所有公式

### 2. 互动式练习题
- **选择题**：精确匹配答案（A、B、C、D）
- **简答题**：关键词匹配，灵活判断
- **编程填空题**：代码片段匹配
- **算法题**：提供测试用例和代码检查

### 3. 即时反馈系统
- ✅ 回答正确显示绿色提示
- ❌ 回答错误显示红色提示
- 支持查看/隐藏标准答案
- 详细的解题思路和解析

### 4. 导航功能
- 按主题分类展示笔记
- 可折叠/展开主题卡片
- 实时搜索过滤（标题、关键词、描述）
- 学习进度追踪（基于 localStorage）

## 📝 文件命名规范

笔记文件采用以下命名格式：
```

[四位数序号]-[主题名称]-[文件标题].html
```
**示例**：
- `0001-数学归纳法-原理与皮亚诺公理.html`
- `0002-数学归纳法-错误证明分析.html`
- `0003-递归与分治-斐波那契数列.html`

**规则**：
- 序号递增：每次新建笔记，序号 +1
- 主题分类：按算法主题分类
- 格式统一：用连字符 `-` 分隔各部分

## 🔄 添加新笔记流程

### 步骤 1：创建新 HTML 文件
在 `pages/` 文件夹下创建新文件，确保包含：
- MathJax 配置（支持单美元符号）
- 互动式 CSS 样式
- JavaScript 功能（答案检查、反馈显示）
- 完整的 LaTeX 公式

### 步骤 2：更新导航页面
编辑 `index.html`，在对应主题的 `<div class="note-list">` 中添加新条目：

```html
<a href="pages/[序号]-[主题]-[标题].html" class="note-item" 
 data-title="标题" data-keywords="关键词1,关键词2">
 <div class="note-number">序号</div>
 <div class="note-content">
 <div class="note-title">标题</div>
 <div class="note-desc">简短描述</div>
 </div>
 <span class="progress-badge new">最新</span>
 <span class="note-arrow">→</span>
</a>
```

### 步骤 3：更新统计数据

- 更新顶部统计区域（笔记数量 +1）
- 更新对应主题的笔记数量

### 步骤 4（可选）：添加新主题

如果是新主题，在 `index.html` 中添加新的主题卡片
